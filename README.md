## Setup

Убедитесь в том, что установлены зависимости:
(full install)
```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Запустите сервер разработки на http://localhost:3000

```bash
npm run dev
```

## Production

Cбилдить нужно 

```bash
npm run build
```

Запустить билд

```bash
npm run preview
```
